# Estruturas_lineares_11242402037
Atividades solicitadas para matéria de estruturas lineares de dados 

## Exercícios

### Exercício 1
Soma dos valores nas posições pares de um vetor.

### Exercício 2
Validação de números pares e ímpares conforme posição no vetor.

### Exercício 2-1
Implementação de uma lista de caracteres com alocação estática (array), incluindo operações de adicionar, remover, concatenar, verificação de palíndromo e codificação.

### Exercício 2-2
Manipulação de lista utilizando ArrayList com operações de inserção, substituição, busca e remoção.

### Exercício 3-1
Mapeamento de país → capital utilizando Map e exibição com forEach.

### Exercício 3-2
Mapeamento de cidade → população utilizando TreeMap com ordenação automática.

### Exercício 3-3
Mapeamento de filme → ano de lançamento utilizando LinkedHashMap mantendo a ordem de inserção.
